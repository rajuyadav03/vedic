# Vaidik Niramay - Ayurveda & Neurotherapy Platform

Welcome to the Vaidik Niramay project! This is a modern, responsive web platform designed for traditional Ayurvedic healing, Neurotherapy services, and academic courses.

## 🌟 Overview

Vaidik Niramay offers a spiritual and seamless digital experience combining ancient Indian healing traditions with a contemporary user interface. The project includes various portals for users, patients, and doctors, all centralized through a main project hub.

## 🚀 Features

- **Fully Responsive Design**: All pages are optimized for mobile, tablet, and desktop viewing.
- **Modern UI/UX**: Implements a sleek "glassmorphism" aesthetic with vibrant, thematic colors (saffron/orange and deep slate).
- **Extensive Typography & Icons**: Uses `Inter` Google font and Google Material Symbols.
- **Centralized Hub**: An easy-to-use navigation dashboard to access all screens within the project.

## 🛠️ Technology Stack

- **HTML5**: Semantic markup focusing on accessibility.
- **Tailwind CSS (via CDN)**: For rapid UI development, utility-first styling, and complex responsive grid layouts.
- **Google Fonts & Material Symbols**: Custom typography and iconography.

## 📁 Project Structure

The project consists of multiple independent HTML screens, connected via the main hub:

- `/index.html` - **Main Hub Dashboard**: Central portal linking to all other pages.
- `/doctor-dashboard/index.html` - **Doctor Dashboard**: Customized administration panel for senior Ayurvedic doctors to manage appointments and patients.
- `/stitch/code.html` - **Sacred Healing**: Landing page focusing on the legacy of Dr. Lajpatrai Mehra and holistic treatments.
- `/stitch (1)/code.html` - **Ayurvedic Essentials (Store)**: E-commerce layout for pure Ayurvedic products with filtering and category navigation.
- `/stitch (2)/code.html` - **Neurotherapy Academy**: Portal for academic courses, certifications, and workshops.
- `/stitch (3)/code.html` - **Admin Control Center**: The original base administration dashboard design.
- `/stitch (4)/code.html` - **Ancient Healing**: Focused landing page merging Vedic sciences with modern neurotherapy.
- `/stitch (5)/code.html` - **Ayurvedic Shop**: Standard product display for the apothecary.

*Note: Directory names (like `stitch (x)`) are preserved from the original Stitch component generation process.*

## 💻 How to Run Locally

Since this project consists of static HTML and CSS files, no complex build process or backend server is required to view the UI.

1. **Clone the repository:**
   ```bash
   git clone https://github.com/rajuyadav03/vedic.git
   ```
2. **Navigate to the project folder:**
   ```bash
   cd vedic
   ```
3. **Open the project:**
   Simply double-click the `index.html` file in the root directory to open the Main Hub in your default web browser. From there, you can navigate to all other screens.
   
   *For the best development experience, consider using a local development server like VS Code's "Live Server" extension.*

## 📱 Mobile Responsiveness

The entire interface has been upgraded with a mobile-first philosophy. Key responsive features include:
- Collapsible hamburger navigation menus on smaller screens for cleaner headers.
- Product and course grid layouts that smoothly transition from multi-column displays on desktop to stacked single or dual columns on mobile.
- Adjusted typography, padding, and hero section heights to ensure clear readability and functional touch targets on mobile devices.
